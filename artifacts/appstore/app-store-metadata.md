## Short Description (80 chars)
AI 기반 어린이 학원 셔틀버스 공유 플랫폼. 실시간 위치 추적과 안전 관리.

## Long Description
SAFEWAY KIDS는 학부모, 학원, 기사, 안전도우미를 연결하는 AI 기반 어린이 통학 안전 플랫폼입니다.

주요 기능:
- 실시간 GPS 위치 추적: 자녀가 탑승한 차량의 위치를 실시간으로 확인
- 탑승/하차 알림: 자녀의 탑승 및 하차 시 즉시 푸시 알림
- AI 최적 경로: 교통 상황을 고려한 최적 운행 경로 자동 생성
- SOS 긴급 호출: 긴급 상황 시 원터치 SOS 기능
- 안전 체크리스트: 세림이법 준수 차량 잔류 확인 절차
- 기사/차량 정보: 배정된 기사 및 차량 정보 투명하게 공개

학부모 기능:
- 자녀 등록 및 스케줄 관리
- 실시간 차량 위치 확인
- 탑승/하차/미탑승 알림 수신

기사 기능:
- 학생 탑승/하차 체크
- AI 최적 경로 네비게이션 연동
- 차량 안전 점검 체크리스트

안전도우미 기능:
- 배정 차량 학생 관리
- 안전 체크리스트 수행
- 출퇴근 및 수당 관리

학원 관리자 기능:
- 학생/스케줄/차량 통합 관리
- 실시간 관제 센터
- 청구/정산 관리

## Notes for Review Team
- This is a children's transportation safety platform
- All users must verify identity via phone OTP
- Children's data is protected under Korean PIPA (Personal Information Protection Act)
- GPS tracking is used only for real-time vehicle location during scheduled rides
- The app does not collect data from children directly; parents manage children's profiles
