### 1.3 "30% 절감" 팩트체크

#### SRS 원문 주장

SRS 2.2절: "셔틀타요 비용 절감 효과: 개별 운행 시세 대비 약 30% 비용 절감"
SRS 3.1절: "차량 유지보수비, 기사 채용 리소스 등 고정비 최대 30% 절감"

#### CFO 분석

**"30% 절감"이 성립하려면 다음 전제가 모두 충족되어야 한다:**

1. **차량 공유가 실제로 작동해야 한다**: A학원과 B학원의 시간대가 어긋나야 차량 공유가 가능. 현실적으로 학원 하원 시간은 대부분 오후 4~6시에 몰린다. 피크 시간 겹침이 심하면 공유율이 급락.

2. **AI 배차 최적화가 공차율을 의미있게 줄여야 한다**: VRP-TW 알고리즘이 구현되어 있어야 하는데, 현재 코드에서 VRP 엔진은 **미구현**. 스케줄 템플릿 UI조차 없음 (이전 학원 리뷰에서 지적).

3. **충분한 네트워크 밀도가 있어야 한다**: 한 동네에 세이프웨이키즈를 쓰는 학원이 3~5곳 이상 밀집해야 교차 탑승이 경제적. 초기에는 네트워크 효과가 0.

**결론: "30% 절감"은 네트워크 밀도가 충분하고, VRP 최적화가 작동하며, 시간대 분산이 이뤄진 이상적 조건에서의 이론적 최대치다. 초기 단계에서는 절감이 0%이거나 오히려 비용이 증가할 수 있다.**

#### 시나리오별 현실적 절감률 추정

| 조건 | 예상 절감률 | 근거 |
|------|-----------|------|
| 초기 (지역 내 학원 1~2곳) | **0% ~ -10%** | 차량 공유 불가, 플랫폼 수수료만 추가 |
| 성장기 (지역 내 5~10곳) | **10~15%** | 일부 차량 공유, 배차 최적화 시작 |
| 성숙기 (지역 내 20곳+) | **20~30%** | 본격적 교차 탑승, 공차율 최소화 |

## 2. 규모별 손익분기점 분석

### 2.1 학원 관점: 자체 운영 vs 세이프웨이키즈

#### 소규모 학원 (30명, 차량 1대)

| 항목 | 자체 운영 | 세이프웨이키즈 |
|------|----------|--------------|
| 차량 관련 비용 | 5,400,000원/월 | 0원 (플랫폼이 운영) |
| 학부모 징수 | 학원이 셔틀비 별도 징수 (약 5만~10만원/명) | 플랫폼이 직접 15만원/명 청구 |
| 학원 순비용 | 5,400,000 - (30 x 70,000*) = 3,300,000원/월 | 0원 (학부모가 직접 플랫폼에 지불) |

*현재 많은 학원이 셔틀비를 월 5만~10만원 수준으로 별도 징수. 평균 7만원 가정.

**학원 입장 절감**: 3,300,000원/월 절감 (차량 운영비 - 셔틀비 수입)
**학부모 입장 부담 변화**: 7만원 → 15만원 (2배 이상 증가!)

> **CFO 코멘트**: "학원 비용은 줄지만 학부모 부담이 2배 된다? 학부모가 수용하겠나? 학부모 이탈 → 학원 이탈 리스크."

#### 중규모 학원 (100명, 차량 3대)

| 항목 | 자체 운영 | 세이프웨이키즈 |
|------|----------|--------------|
| 차량 관련 비용 | 16,200,000원/월 | 0원 |
| 학부모 징수 | 100 x 70,000 = 7,000,000원/월 | 100 x 150,000 = 15,000,000원 (학부모 직접) |
| 학원 순비용 | 9,200,000원/월 | 0원 |

**학원 절감**: 9,200,000원/월 (56.8%)
**학부모 부담 증가**: 70,000원 → 150,000원 (+114%)

#### 대규모 학원 (300명, 차량 8대)

| 항목 | 자체 운영 | 세이프웨이키즈 |
|------|----------|--------------|
| 차량 관련 비용 | 43,200,000원/월 | 0원 |
| 학부모 징수 | 300 x 70,000 = 21,000,000원/월 | 300 x 150,000 = 45,000,000원 (학부모 직접) |
| 학원 순비용 | 22,200,000원/월 | 0원 |

**학원 절감**: 22,200,000원/월 (100% — 셔틀 운영 자체를 아웃소싱)
**학부모 부담 증가**: 70,000원 → 150,000원 (+114%)

### 2.2 핵심 문제: 비용 전가 구조

> **"30% 절감이라고 했는데, 실제로는 학원 비용이 0이 되고 학부모 부담이 2배가 되는 구조다. 이건 '절감'이 아니라 '전가'다."**

현재 빌링 모델의 구조적 특징:
- 학원은 플랫폼에 돈을 내지 않는다
- 학부모가 건당/월 상한으로 플랫폼에 직접 지불한다
- 학원의 셔틀 운영비는 확실히 줄지만, 그 비용이 학부모에게 옮겨간다

**이 구조에서 "30% 절감"은 학원 관점에서만 성립하며, 학부모 관점에서는 비용 증가다.**

## 3. 지입차 전환 비용 분석

### 3.1 전환 비용 항목

우리 프랜차이즈 150개 지점 중 약 120개 지점이 지입차 기사와 계약 중. 전환 시 발생하는 비용:

| 비용 항목 | 지점당 | 120개 지점 합계 |
|-----------|--------|----------------|
| 지입차 계약 해지 위약금 | 1,000,000~3,000,000원 | 1.2~3.6억원 |
| 이중 운영 기간 (2개월) | 10,800,000원 | 12.96억원 |
| 학생/스케줄 데이터 이관 | 500,000원 (인건비) | 6,000만원 |
| 직원 교육 (대시보드 사용) | 200,000원 | 2,400만원 |
| **전환 비용 합계** | | **14.4~17억원** |

### 3.2 기사 반발 리스크

- 지입차 기사는 개인사업자. 일방적 계약 해지 시 생계 위협 → 민원, 법적 분쟁 가능
- 기존 기사를 세이프웨이키즈 플랫폼 기사로 전환할 수 있는 방안이 SRS에 언급되어 있으나, 구체적 조건(수수료율, 수입 보장) 미정
- 기사 노조/연합 대응 필요

### 3.3 전환 기간 이중비용

- 기존 지입차 계약 해지 통보 → 1~3개월 해지 유예기간
- 세이프웨이키즈 도입 → 최소 2주~1개월 세팅 기간 (학원 리뷰 기준: 초기 세팅 반나절~1일)
- 이 기간 동안 기존 비용 + 신규 비용 이중 발생

### 4.3 규제 리스크 비용

SRS 4.1절에 명시: "유상운송의 금지 조항으로 인해 불법 소지가 농후"

- 규제 샌드박스 승인 전까지 법적 회색 지대에서 운영
- 규제 샌드박스 신청 완료 (2026-03-20) 상태이나 승인 시점 미확정
- **승인 실패 시 사업 자체가 불가** → 150개 지점이 일제히 전환했다가 규제 실패하면 다시 지입차로 돌아가야 하는 최악의 시나리오

## 6. 가격 모델 평가 및 대안

### 6.1 현행 모델 평가

**현행**: 건당 5,000원 / 월 상한 150,000원 (학부모 직접 부담)

| 장점 | 단점 |
|------|------|
| 사용량 비례 → 형평성 | 학부모 월 부담 15만원 = 기존 대비 2배 |
| 학원 비용 0 → 도입 장벽 낮음 | 학부모 가격 저항 → 학원이 도입 꺼림 |
| 자동 청구 → 학원 관리 부담 감소 | PG 수수료 2.5~3.5%가 마진 침식 |
| 월 상한으로 예측 가능 | 건당 과금 구조는 탑승 횟수 적은 학생에게 불리 |

**문제점**:
1. 월 15만원은 사교육비 60만원의 25% — 셔틀비 하나에 이 비율은 학부모 저항이 클 것
2. 학원이 비용을 안 내니 학원에 대한 가격 지렛대가 없음 → 학원 이탈 방지 장치 부재
3. 플랫폼 수익 구조가 불명확 — 코드에 플랫폼 수수료 로직이 없음

### 6.2 대안 모델 제안

#### 모델 A: SaaS 월정액 (학원 부담) + 학부모 소액

| 항목 | 금액 |
|------|------|
| 학원 SaaS 월정액 | 500,000~2,000,000원/월 (학생 수 기반 구간제) |
| 학부모 셔틀비 | 70,000~100,000원/월 (기존과 유사) |

- 학원이 기존 자체 운영비(16.2M) 대비 SaaS비(1~2M) + 학부모 징수(7~10M) = 8~12M → **25~50% 실 절감**
- 학부모 부담은 기존과 유사하거나 소폭 증가 → 저항 최소
- 플랫폼은 안정적 구독 수익 확보

#### 모델 B: 하이브리드 (학원 기본료 + 건당 과금 분담)

| 항목 | 금액 |
|------|------|
| 학원 기본료 | 300,000원/월 (플랫폼 사용료) |
| 학부모 건당 | 2,000~3,000원/건 (월 상한 80,000~100,000원) |

- 학원과 학부모가 비용을 분담 → 양측 모두 부담 적정
- 학원 기본료가 있으므로 학원 이탈 방지 계약 가능
- 학부모 월 상한 8~10만원 = 기존과 유사 → 수용 가능

#### 모델 C: 프랜차이즈 본사 계약 (볼륨 디스카운트)

| 항목 | 금액 |
|------|------|
| 본사 일괄 계약 (150개 지점) | 지점당 300,000~500,000원/월 |
| 볼륨 디스카운트 | 50개 지점 이상 시 20% 할인 |
| 학부모 셔틀비 | 80,000~100,000원/월 (기존과 유사) |

- 프랜차이즈 본사가 일괄 협상 → 지점별 개별 도입 대비 효율적
- 본사가 가격을 통일하면 지점 간 형평성 확보
- **CFO 선호 모델**: 본사가 비용을 통제하고 학부모 부담을 최소화

### 6.3 CFO 추천

> **"모델 C가 우리한테 맞다. 본사가 플랫폼과 직접 계약하고, 지점에는 가이드라인만 내린다. 학부모 셔틀비는 기존 수준(7~10만원)을 유지해야 한다. 15만원은 절대 안 된다."**

## 8. 종합 평가

### 8.1 재무적 평가 점수

| 항목 | 점수 | 비고 |
|------|------|------|
| 비용 절감 근거 | 25/100 | "30%"의 산출 근거 없음, 이론적 최대치를 마케팅에 사용 |
| 가격 모델 적정성 | 35/100 | 학부모 비용 2배 = 시장 수용성 의문 |
| 프랜차이즈 대응력 | 20/100 | 볼륨 디스카운트, 본사 계약, 다지점 관리 없음 |
| 재무 투명성 | 40/100 | 청구서 자동 생성은 좋으나 수수료율/마진 비공개 |
| 전환 비용 대비 ROI | 30/100 | 초기 17억원 전환 비용 대비 ROI 입증 불가 |
| 규제 리스크 관리 | 35/100 | 샌드박스 신청했으나 미승인, 대안 없음 |
| **종합** | **31/100** | |

### 8.2 CFO 최종 의견

> "솔직히 현재 상태로는 경영진에게 도입을 추천할 수 없다. 이유는 3가지다."

**1. 가격 구조가 잘못되어 있다**
학원 비용을 학부모에게 전가하는 모델은 학원 원장은 좋아하겠지만, 학부모 반발로 결국 학원도 피해를 본다. B2B SaaS + 학부모 소액 분담 모델로 전환해야 한다.

**2. "30% 절감"은 마케팅 슬로건이지 재무적 사실이 아니다**
VRP 배차 최적화가 미구현이고, 네트워크 밀도가 0인 상태에서 30% 절감은 불가능하다. 초기에는 오히려 비용이 증가한다. 파일럿에서 실 데이터를 확보한 후 절감률을 주장해야 한다.

**3. 규제 리스크가 해결되지 않았다**
150개 지점을 전환했다가 규제 샌드박스가 불승인되면 다시 원복해야 한다. 샌드박스 승인 확정 전에는 파일럿 2~3개 지점으로 제한해야 한다.

### 8.3 도입 조건 (이 조건이 충족되면 재검토)

1. 규제 샌드박스 **승인 확정**
2. 파일럿 운영 **실 데이터** (최소 3개월, 3개 지점)
3. **학부모 비용 10만원 이하** 가격 모델 제시
4. **프랜차이즈 본사 계약** 제도 (볼륨 디스카운트, 일괄 관리)
5. **SLA 보장** (장애 시 대체 운송, 보상 규정)
6. 기존 **기사 전환 프로그램** 구체안
7. **VRP 배차 최적화** 실 구현 및 절감률 실증
