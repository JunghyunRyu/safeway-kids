## 4. 잔여 리스크

- **#65 TTS:** `expo-speech`는 Expo Go에서 동작하나, 일부 안드로이드 기기에서 한국어 TTS 엔진 미설치 시 무음 가능. 런타임 에러는 catch 처리됨.
- **#68 ERP Webhook:** 현재 webhook 등록/관리만 구현. 실제 이벤트 발생 시 webhook 발송 로직은 향후 구현 필요 (발행-구독 패턴).
- **#72 SSG:** SPA 셸 복사 방식. 메타태그 주입(Open Graph 등)은 향후 강화 가능.
- **#77 배차 알림:** FCM 토큰 미등록 기사에게는 알림 미발송 (silent skip, 로그 기록).
- **웹 테스트:** 초기 실행 시 1건 간헐 실패 관측 (재실행 시 12/12 통과, 환경 초기화 타이밍 이슈로 추정).
