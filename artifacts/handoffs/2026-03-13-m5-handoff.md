## Demo Quick Start
```bash
# Start infra
docker-compose up -d  # PostgreSQL + Redis

# Backend
cd backend
alembic upgrade head
python -m app.seed       # Seed demo data + generate routes
uvicorn app.main:app     # Start API

# Mobile
cd mobile
npx expo start           # Start Expo dev server
```
