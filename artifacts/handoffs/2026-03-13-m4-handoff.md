## Architecture — End-to-End GPS Flow
```
Driver App                    Backend                         Parent App
─────────                    ───────                         ──────────
useGpsTracking              POST /telemetry/gps              useVehicleTracking
  → POST every 5s   ──→     update_gps():                    ← ws://...?token=JWT
                             1. Redis SET (latest)
                             2. Redis PUBLISH ──────────────→ WebSocket handler
                             3. Redis RPUSH (buffer)            (JWT verified)
                                    │
                             gps_flush_loop() (30s)
                                    │
                             flush_gps_buffer()
                                    ↓
                             PostgreSQL gps_history
```
